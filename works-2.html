MetaLife（架空ホームページ）

「MetaLife」を題材に制作した架空のサービスホームページです。
サービスの魅力をシンプルかつ直感的に伝えられるよう、横スクロール形式で構成しました。

こちらからご覧いただけます

担当

デザイン構成・コーディング

サイトの目的

サービス紹介を通じてユーザーの理解を深め、問い合わせや利用につなげることを想定しています。
SNSやプロフィールリンク先として利用できる、シンプルなHPをイメージしました。

使用技術

HTML / CSS / JavaScript / jQuery

デザインについて

スクロールを最小限に抑え、横に流れる動きで印象に残るよう設計しました。
短時間でサービスの全体像を把握できる、ユーザーフレンドリーな構成を意識しています。

コーディングについて

レスポンシブ対応を行い、シンプルで軽量なコードを心がけました。
アニメーションや動的要素を追加し、より魅力的な表現を心がけました。
